## 一、单选题（共5小题，每小题2分，共10分）

1. 以下函数（其中 $C_1, C_2, C_3$ 为任意常数）中为微分方程 $y''' - 4y' = 0$ 通解的是（）。
(A) $y = C_1 e^{-2x} + C_2 e^{2x} + C_3$
(B) $y = C_1 e^{-2x} + C_2 e^{2x} + C_3 x$
(C) $y = C_1 e^{-2x} + C_2 x e^{2x} + C_3$
(D) $y = C_1 e^{-2x} + C_2 x e^{2x} + C_3 x$

2. 已知二元函数 $\displaystyle f(x, y) = \frac{x^2 y^2}{x^2 y^2 + (x - y)^2}$，则（）。
(A) 二重极限 $\displaystyle \lim_{\substack{x \to 0 \\ y \to 0}} f(x, y)$ 存在，且 $\displaystyle \lim_{x \to 0} \lim_{y \to 0} f(x, y) = 0$
(B) 二重极限 $\displaystyle \lim_{\substack{x \to 0 \\ y \to 0}} f(x, y)$ 存在，且 $\displaystyle \lim_{x \to 0} \lim_{y \to 0} f(x, y) = 1$
(C) 二重极限 $\displaystyle \lim_{\substack{x \to 0 \\ y \to 0}} f(x, y)$ 不存在，但 $\displaystyle \lim_{x \to 0} \lim_{y \to 0} f(x, y) = 0$
(D) 二重极限 $\displaystyle \lim_{\substack{x \to 0 \\ y \to 0}} f(x, y)$ 不存在，但 $\displaystyle \lim_{x \to 0} \lim_{y \to 0} f(x, y) = 1$

3. 方程 $x^2 - 2y^2 + 3z^2 = 0$ 所表示的曲面为（）。
(A) 椭圆柱面  
(B) 椭圆锥面  
(C) 单叶双曲面  
(D) 双叶双曲面

4. 设 $D$ 是由 $x = -1, \ y = 1, \ y = x^3$ 所围成的平面闭区域，$D_1$ 是 $D$ 在第一象限的部分，则二重积分 $\displaystyle \iint_{D} (xy + \cos x \sin y) \, \mathrm{d}x\mathrm{d}y$ 等于（）。
(A) $0$
(B) $\displaystyle 2 \iint_{D_1} xy \, \mathrm{d}x\mathrm{d}y$
(C) $\displaystyle 4 \iint_{D_1} (xy + \cos x \sin y) \, \mathrm{d}x\mathrm{d}y$
(D) $\displaystyle 2 \iint_{D_1} \cos x \sin y \, \mathrm{d}x\mathrm{d}y$

5. 设函数 $f(x)$ 是 $(-\infty, \infty)$ 上以 $2\pi$ 为周期的周期函数，它在 $[-\pi, \pi)$ 上的表达式为
$$f(x) = \begin{cases}
x, & -\pi \leq x < 0 \\
0, & 0 \leq x < \pi
\end{cases}$$
其傅里叶级数 $\displaystyle \frac{a_0}{2} + \sum_{n=1}^{\infty} (a_n \cos nx + b_n \sin nx)$ 的和函数为 $S(x)$，则 $a_0$ 和 $S(3\pi)$ 分别为（）。

(A) $\displaystyle \frac{\pi}{2}, -\frac{\pi}{2}$
(B) $\displaystyle -\frac{\pi}{2}, \frac{\pi}{2}$
(C) $\displaystyle -\frac{\pi}{2}, -\frac{\pi}{2}$
(D) $\displaystyle \frac{\pi}{2}, \frac{\pi}{2}$

## 三、解答题（共11小题，共80分）

11.（6分）问当 $t$ 为何值时，空间四点 $A(1, 0, 0), \ B(0, 2, 0), \ C(0, 0, 3), \ D(-1, 2, t)$ 共面？并求平面四边形 $ABCD$ 的面积。

12.（6分）设 $z = f(x - y) + g(x^2 y, 3x - 4y)$，其中 $f$ 具有连续的二阶导数，$g$ 具有连续的二阶偏导数，求 $\displaystyle \frac{\partial^2 z}{\partial x \partial y}$。

13.（6分）计算二重积分 $\displaystyle \iint_{D} (x + y)^2 \, d\sigma$，其中 $D = \{(x, y) \mid x^2 + y^2 \leq x\}$。

14.（6分）已知物体在空间所占区域为 $\Omega = \{(x, y, z) \mid 1 \leq x^2 + y^2 + z^2 \leq 2\}$，其密度函数为 $f(x, y, z) = \sqrt{x^2 + y^2}$，求该物体的质量。

15.（6分）问：当常数 $a$ 为何值时，存在可微函数 $u = u(x, y)$，使得
$$du = (x - 2y + 1) \mathrm{d}x + (ax + y^2 + 3) \mathrm{d}y?$$
若 $u(0, 0) = 0$，求函数 $u(x, y)$ 的表达式。

16.（6分）求幂级数 $\displaystyle \sum_{n=0}^{\infty} \frac{n}{2n + 1} x^{2n + 1}$ 的收敛域与和函数。

17.（8分）已知 $\boldsymbol{u}$ 是曲线
$$\begin{cases}
x + y + z = 0 \\
x^3 + y^3 - z^3 = 10
\end{cases}$$
在点 $P(1, 1, -2)$ 处与 $y$ 轴正向夹角为锐角的切线方向，求函数 $f(x, y, z) = \ln(1 + 2xy + yz)$ 在点 $P$ 处沿方向 $\boldsymbol{u}$ 的方向导数。

18.（8分）设点 $P$ 为曲面 $S: F(x, y, z) = 0$ 上离原点最近的点，其中 $F(x, y, z)$ 具有连续的偏导数，证明：曲面 $S$ 在点 $P$ 处的法线必过原点。

19.（8分）计算曲线积分
$$\oint_{\Gamma} (x + y - y^2) \mathrm{d}x + (z + x - x^2)\mathrm{d}z$$
其中曲线 $\displaystyle \begin{cases} x^2 + y^2 + z^2 = 4 \\ z = x \end{cases}$ 从 $z$ 轴的正向看去，其方向为逆时针方向。

20.（10分）求流速场 $\boldsymbol{v} = (x^2 - xy, y^2 - yz, z^2 - zx)$ 朝外穿过曲面 $\Sigma$ 的流量，其中 $\Sigma$ 是由曲线
$$\begin{cases}
z = \sqrt{y - 1}, & (1 \leq y \leq 3) \\
x = 0
\end{cases}$$
绕 $y$ 轴旋转一周所得的旋转曲面。

21.（10分）在研究某种病毒在孤立环境下传播时，把人群分成未感染者（健康人）和已感染者（病人）两类。当健康人与病人有效接触后受感染变成病人；病人治愈成为健康人后，也可再次被感染。设该环境下人群总数为常数 $N$，在 $t$ 时刻健康人数和病人数分别为 $S(t)$ 和 $I(t)$（将 $S(t)$ 和 $I(t)$ 视为 $t$ 的连续可微函数）。假定：

I) 每天健康人受感染成为病人的人数与 $S(t) I(t)$ 成正比，比例系数为 $\lambda$（称 $\lambda$ 为接触率，$\lambda > 0$）；

II) 每天被治愈的病人数与 $I(t)$ 成正比，比例系数为 $\mu$（称 $\mu$ 为治愈率，$\mu > 0$）。

已知 $I(0) = I_0$。

(1) 试建立函数 $I(t)$ 的微分方程，并求解此方程；（7分）

(2) 问 $\lambda, \mu$ 满足什么条件时，$\displaystyle \lim_{t \to \infty} I(t) = 0$？并据此对防疫提出有益建议。（3分）
