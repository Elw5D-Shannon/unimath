## 一、单选题（共5小题，每小题2分，共10分）

1. 下列函数在点 $(0,0)$ 处可微的是（）。
(A) $\sqrt{x^2 + y^4}$  
(B) $\sqrt{x^4 + y^4}$  
(C) $\sqrt{x^2 + y^6}$  
(D) $\sqrt{x^2 + y^2}$

2. 已知向量值函数 $\boldsymbol{r}(t) = (t, t^2, t^3 - t) \ (t > 0)$ 表示某运动质点在时刻 $t$ 的位置，则在运动过程中速度与加速度相互垂直的时刻有（）个。
(A) 0  
(B) 1  
(C) 2  
(D) 3

3. 已知函数 $f(x, y) = \ln(1 + xy) + y \cos x$，则 $f(x, y)$ 在点 $(0,1)$ 处沿任意方向的方向导数都不可能取到的值是（）。
(A) 0  
(B) $-1$  
(C) 1  
(D) 2

4. 函数 $f(x, y) = x^3 - y^3 + 3x^2 + 3y^2 - 9x + 7$ 的极大值点为（）。
(A) $(-3, 2)$  
(B) $(-3, 0)$  
(C) $(1, 0)$  
(D) $(1, 2)$

5. 下列正向闭曲线中，使曲线积分 $\displaystyle \oint_{L} \frac{y \,\mathrm{d}x - x \,\mathrm{d}y}{x^2 + xy + y^2} = 0$ 的曲线是（）。
(A) $L: x^2 + y^2 = 1$
(B) $L: x^2 + xy + y^2 = 1$
(C) $L: (x - 1)^2 + (y - 1)^2 = 1$
(D) $L: x^{\frac{2}{3}} + y^{\frac{2}{3}} = 1$

## 三、解答题（共11小题，共80分）

11.（6分）求曲面 $\displaystyle S: z = e^{-x} \sin \frac{x}{y}$ 在点 $(0, 1, 0)$ 处的切平面方程和法线方程。

12.（6分）已知函数 $z = z(x, y)$ 由方程 $z = f(x, x - 2y) + g(z)$ 所确定，其中 $f(u, v)$ 具有连续的一阶偏导数，$g(t)$ 具有连续的一阶导数且 $g'(t) \neq 1$，求 $\displaystyle \frac{\partial z}{\partial x}, \frac{\partial z}{\partial y}$。

13.（6分）已知平面薄片所占区域为 $D = \{(x, y) \mid y \leq x^2 + y^2 \leq 2y, \ x \geq 0\}$，其面密度为 $f(x, y) = \sqrt{x^2 + y^2}$，求该薄片的质量。

14.（6分）计算二重积分 $\displaystyle \iint_{D} \frac{1}{\sqrt{1 + y^3}} \,\mathrm{d}\sigma$，其中 $D$ 是由曲线 $x = y^2$，直线 $y = 2$ 及 $y$ 轴所围成的闭区域。

15.（6分）已知定点 $P$ 到定平面 $\pi$ 的距离为 $a \ (a > 0)$，试建立合适坐标系，求到定点 $P$ 和到定平面 $\pi$ 的距离之比为常数 $k \ (k > 0)$ 的动点的轨迹方程，并指出其几何形状。

16.（6分）设三元函数 $f(x, y, z)$ 连续，$f(x, y, z) \neq 0$ 且满足
$$f(x, y, z) \iiint_{\Omega} f(x, y, z) \,\mathrm{d}V = (x^2 + y^2 + z^2) \unicode{0x222F}_{\Sigma} f(x, y, z) \,\mathrm{d}S$$
其中 $\Omega$ 为球面 $\Sigma: x^2 + y^2 + z^2 = 1$ 所围成的闭区域，求 $f(x, y, z)$ 的表达式。

17.（8分）已知函数 $\varphi(x)$ 可导且 $\varphi(0) = 1$，试确定函数 $\varphi(x)$ 使得曲线积分
$$\int_{L} \left[xy^2 \varphi(x) + e^x\right]\mathrm{d}x + \left[y \varphi(x) + \frac{1}{1 + y^2}\right]\mathrm{d}y$$
与积分路径无关，并计算曲线积分
$$\int_{(0,0)}^{(1,1)} \left[xy^2 \varphi(x) + e^x\right]\mathrm{d}x + \left[y \varphi(x) + \frac{1}{1 + y^2}\right]\mathrm{d}y$$

18.（8分）求幂级数 $\displaystyle \sum_{n=0}^{\infty} \frac{n^2 + 1}{n!} x^n$ 的收敛域与和函数，并求级数 $\displaystyle \sum_{n=0}^{\infty} \frac{n^2 + 1}{2^n \cdot n!}$ 的和。

19.（8分）设 $\Sigma$ 为 $yOz$ 平面上的曲线 $z = e^y \ (0 \leq y \leq 1)$ 绕 $z$ 轴旋转成的曲面，其法向量指向上侧，计算曲面积分
$$I = \iint_{\Sigma} 4xz \,\mathrm{d}ydz + (y - 2yz) \,\mathrm{d}zdx + (1 - z^2) \,\mathrm{d}xdy$$

20.（10分）已知 $y_1 = e^x + x$，$y_2 = 2e^{-x} + x$，$y_3 = e^x + e^{-x} + x$ 是某二阶常系数线性微分方程 $y'' + py' + qy = f(x)$ 的三个特解。

(1) 试写出该方程的具体形式，并求此方程的通解；（6分）

(2) 求方程 $y'' + py' + qy = f(x) + xe^x$ 的通解。（4分）
