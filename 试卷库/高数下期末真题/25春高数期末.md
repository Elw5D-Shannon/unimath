
## 一、单选题（共5小题，每小题2分，共10分）

1. 点 $P(1, -1,2)$ 关于 $y$ 轴的对称点位于（ ）  
   （A）第I卦限  
   （B）第III卦限  
   （C）第V卦限  
   （D）第VI卦限

2. 函数 $z = \mathrm{e}^{-x}(\cos y + \sin y)$ 在点 $(0,0)$ 处的微分 $\mathrm{d}z\big|_{(0,0)} =$（ ）  
   （A）$-dx - dy$  
   （B）$-dx + dy$  
   （C）$dx - dy$  
   （D）$dx + dy$

3. 设幂级数 $\sum_{n = 0}^{\infty}a_{n}(x + 1)^{n}$ 在 $x = -3$ 处条件收敛，则下列四个集合  
   $$
   ①\{-2, -1,0\} \qquad ②\{-1,0,1\} \qquad ③\{1,2,3\} \qquad ④\{-1,0,3\}
   $$  
   必是幂级数 $\sum_{n = 0}^{\infty}a_{n}(x - 1)^{n}$ 的收敛域的子集的个数为（ ）  
   （A）1  
   （B）2  
   （C）3  
   （D）4

4. 已知曲线 $C_1:x^2 +y^2 = 1$，$C_2:x^2 +y^2 = 2x$ 和 $C_3:|x| + |y| = 1$，记  
   $$
   I_{k} = \oint_{C_{k}}(x^{2} + xy + y^{2})\,\mathrm{d}s \quad (k = 1,2,3),
   $$  
   则（ ）  
   （A）$I_{1}< I_{2}< I_{3}$  
   （B）$I_{2}< I_{3}< I_{1}$  
   （C）$I_{3}< I_{1}< I_{2}$  
   （D）$I_{3}< I_{2}< I_{1}$

5. 已知函数 $f(x) = [x]$, $x \in [0,2)$, 其中 $[x]$ 为不超过 $x$ 的最大整数。设  
   $$
   a_{n} = \int_{0}^{2}f(x)\cos n\pi x\,\mathrm{d}x \quad (n = 0,1,2,\dots), \qquad
   b_{n} = \int_{0}^{2}f(x)\sin n\pi x\,\mathrm{d}x \quad (n = 1,2,\dots),
   $$  
   令  
   $$
   S(x) = \frac{a_{0}}{2} +\sum_{n = 1}^{\infty}(a_{n}\cos n\pi x + b_{n}\sin n\pi x),
   $$  
   则 $b_{1}$ 和 $S(2025)$ 的值分别为（ ）  
   （A）$-\frac{2}{\pi},0$  
   （B）$-\frac{2}{\pi},\frac{1}{2}$  
   （C）$\frac{2}{\pi},0$  
   （D）$\frac{2}{\pi},\frac{1}{2}$

## 三、解答题（共11小题，共80分）

11. （6分）已知函数 $z = f(x,y)$ 的梯度为 $\nabla f(x,y) = (2x + y,\ x - y)$。  
    (1) 求函数 $f(x,y)$ 在点 $A(1,2)$ 处沿从该点到点 $B(4, -2)$ 的方向的方向导数。（3分）  
    (2) 若 $f(0,0) = 0$，求 $f(x,y)$ 的表达式。（3分）

12. （6分）设函数 $z = z(x,y)$ 由方程 $f(ax + z,\ by + z) = 0$ 所确定，其中 $a,b$ 为正常数，$f(u,v)$ 具有二阶连续偏导数，且 $f_{u}^{\prime} + f_{v}^{\prime}\neq 0$。  
    (1) 求 $b\frac{\partial z}{\partial x} + a\frac{\partial z}{\partial y}$。（3分）  
    (2) 证明：$\left(\frac{\partial^{2}z}{\partial x\partial y}\right)^{2} = \frac{\partial^{2}z}{\partial x^{2}}\cdot \frac{\partial^{2}z}{\partial y^{2}}$。（3分）

13. （6分）计算二重积分 $\iint_{D}\frac{x^{2}}{1 + (y - 1)^{4}}\,\mathrm{d}\sigma$，其中 $D$ 是由直线 $y = 1 - x$，$y = -1$ 及 $y$ 轴所围成的闭区域。

14. （6分）计算三重积分 $\iiint_{\Omega}(y + z)\,\mathrm{d}V$，其中 $\Omega$ 为曲面 $z = x^{2} + y^{2}$ 和平面 $z = x$ 所围成的空间闭区域。

15. （6分）求函数 $f(x, y) = x^{3} + y^{3} - x^{2} - 2xy - y^{2} + x + y + 3$ 的极值。

16. （6分）计算曲线积分  
    $$
    I = \oint_{L}(y + \sin x)\,\mathrm{d}x + (z + \sin y)\,\mathrm{d}y + (x + \sin z)\,\mathrm{d}z,
    $$  
    其中 $L$ 是圆柱面 $x^{2} + y^{2} = 1$ 与平面 $z = x + y$ 的交线，从 $z$ 轴正向看去，$L$ 为逆时针方向。

17. （8分）求微分方程 $y'' + y = \sin x + x$ 的通解。

18. （8分）计算曲线积分  
    $$
    I = \int_{C}(\mathrm{e}^{y} - y + 1)\,\mathrm{d}x + (x\mathrm{e}^{y} + y + 1)\,\mathrm{d}y,
    $$  
    其中 $C$ 为从原点 $O$ 沿曲线 $y = |\sin x|$ 到点 $A(2\pi, 0)$ 的有向弧段。

19. （8分）设有流速场  
    $$
    \nu(x,y,z) = \left(
    \frac{b y - c z}{a x^{2} + b y^{2} + c z^{2}},
    \frac{c z - a x}{a x^{2} + b y^{2} + c z^{2}},
    \frac{a x - b y}{a x^{2} + b y^{2} + c z^{2}}
    \right),
    $$  
    其中 $a,b,c$ 为常数，且 $a^{2} + b^{2} + c^{2}\neq 0$。  
    (1) 证明 $\mathrm{div}\,\nu = 0$；（3分）  
    (2) 计算曲面积分  
    $$
    \oint_{\Sigma}\left(
    \frac{b y - c z}{a x^{2} + b y^{2} + c z^{2}} + x
    \right)\mathrm{d}y\mathrm{d}z
    + \left(
    \frac{c z - a x}{a x^{2} + b y^{2} + c z^{2}} + y
    \right)\mathrm{d}z\mathrm{d}x
    + \left(
    \frac{a x - b y}{a x^{2} + b y^{2} + c z^{2}} + z
    \right)\mathrm{d}x\mathrm{d}y,
    $$  
    其中 $\Sigma$ 为球面 $x^{2} + y^{2} + z^{2} = 1$ 的外侧。（5分）

20. （10分）设数列 $\{a_{n}\}$ 满足 $a_{1} = 1$，$a_{n + 1} = \frac{2n + 1}{2n + 2} a_{n}$，$n = 1,2,3,\dots$。  
    (1) 证明：当 $|x| < 1$ 时，幂级数 $\sum_{n = 1}^{\infty} a_{n} x^{n}$ 收敛；（3分）  
    (2) 求数值级数 $\sum_{n = 1}^{\infty} \frac{a_{n}}{2^{n}}$ 的和。（7分）

21. （10分）2025年春节档，国产动画片《哪吒之魔童闹海》备受瞩目，登顶全球动画电影票房榜。剧中“天元鼎”的设计灵感源于我国古代青铜器，彰显出中华传统文化的深厚底蕴。现给出“天元鼎”的一个简化造型 $\Sigma$，它由上、下两个球面片 $\Sigma_{1}$ 和 $\Sigma_{2}$ 对接围成，其中  
    $$
    \Sigma_{1}: x^{2} + y^{2} + z^{2} = \frac{8}{3} z \quad \left(\frac{3}{2}\leq z\leq \frac{8}{3}\right),
    $$  
    $$
    \Sigma_{2}: x^{2} + y^{2} + z^{2} = 4 \quad \left(-2\leq z\leq \frac{3}{2}\right).
    $$  
    (1) 计算 $\Sigma$ 的表面积 $S$。（5分）  
    (2) 假设南极仙翁悬浮于空中一点 $P(1,2,1)$ 处，请帮助哪吒在 $\Sigma$ 的下部 $\Sigma_{2}$ 上寻找一个点 $M$，使他能以最短距离攻击南极仙翁。试写出点 $M$ 的坐标。（5分）
