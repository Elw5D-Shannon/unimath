
## 一、单选题（共5小题，每小题2分，共10分）

1. 曲面 $S: 2x^2 - 3y^2 + z^2 + 4x + 6y = 0$ 的图形是（）。
(A) 双叶双曲面  
(B) 单叶双曲面  
(C) 椭圆锥面  
(D) 椭圆抛物面

2. 已知 $z = z(x, y)$ 是由方程 $f(x + az, y + bz) = 0$ 所确定的隐函数，则（）。
(A) $\displaystyle a \frac{\partial z}{\partial x} + b \frac{\partial z}{\partial y} = 1$
(B) $\displaystyle a \frac{\partial z}{\partial x} + b \frac{\partial z}{\partial y} = -1$
(C) $\displaystyle b \frac{\partial z}{\partial x} + a \frac{\partial z}{\partial y} = 1$
(D) $\displaystyle b \frac{\partial z}{\partial x} + a \frac{\partial z}{\partial y} = -1$

3. 微分方程 $y'' - y' = x(e^x + 1)$ 的特解形式可设为（）。
(A) $(ax + b)e^x + cx + d$
(B) $(ax^2 + bx)e^x + cx + d$
(C) $(ax + b)e^x + cx^2 + dx$
(D) $(ax^2 + bx)e^x + cx^2 + dx$

4. 已知幂级数 $\displaystyle \sum_{n=0}^{\infty} a_n(x + 2)^n$ 在 $x = -4$ 处条件收敛，则该幂级数（）。
(A) 在 $x = -1$ 和 $x = 1$ 处都收敛  
(B) 在 $x = -1$ 处发散，在 $x = 1$ 处收敛  
(C) 在 $x = -1$ 和 $x = 1$ 处都发散  
(D) 在 $x = -1$ 处收敛，在 $x = 1$ 处发散

5. 已知函数$f(x) = \begin{cases}-1, & 0 < x \leq \frac{\pi}{2} \\2, & \frac{\pi}{2} < x < \pi\end{cases}$，$S(x)$ 为正弦级数 $\displaystyle \sum_{n=1}^{\infty} b_n \sin nx$ 的和函数，其中 $\displaystyle b_n = \frac{2}{\pi} \int_{0}^{\pi} f(x) \sin nx \, dx \ (n = 1, 2, \cdots)$，则 $S\left(-\frac{9\pi}{2}\right)$ 与 $S\left(\frac{9}{2}\right)$ 的值分别为（）。
(A) $\displaystyle \frac{1}{2}, -2$
(B) $\displaystyle \frac{1}{2}, -1$
(C) $\displaystyle -\frac{1}{2}, -2$
(D) $\displaystyle -\frac{1}{2}, -1$

(2) 求开伞后减速下落的时间。（2分）
