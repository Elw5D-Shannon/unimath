## 一、单选题（共5小题，每小题3分，共15分）

1. 曲线 $C: x = 1 + t, \ y = 1 + t^2, \ z = 1 + t^3$ 在点 $P(0, 2, 0)$ 处的切线方程为（）。
(A) $\displaystyle \frac{x}{1} = \frac{y - 2}{0} = \frac{z}{0}$
(B) $\displaystyle \frac{x}{1} = \frac{y - 2}{2} = \frac{z}{3}$
(C) $\displaystyle \frac{x}{1} = \frac{y - 2}{-2} = \frac{z}{3}$
(D) $\displaystyle \frac{x - 1}{1} = \frac{y - 1}{-2} = \frac{z - 1}{3}$

2. 已知 $f(x)$ 是连续函数，则二次积分 $\displaystyle \int_{0}^{1}\mathrm{d}x \int_{x}^{1} f(x) f(y)\mathrm{d}y$ 等于（）。
(A) $\displaystyle \int_{0}^{1} f(x)\mathrm{d}x$
(B) $\displaystyle \frac{1}{2} \int_{0}^{1} f(x)\mathrm{d}x$
(C) $\displaystyle \left(\int_{0}^{1} f(x)\mathrm{d}x\right)^2$
(D) $\displaystyle \frac{1}{2} \left(\int_{0}^{1} f(x)\mathrm{d}x\right)^2$

3. 已知函数 $z = f(x, y)$ 具有连续的一阶偏导数，$P(x_0, y_0, z_0)$ 为曲面 $S: z = f(x, y)$ 上一点（如图所示），则（）。

## 三、解答与证明题（共10小题，共70分）

11.（6分）已知 $z = z(x, y)$ 是由方程 $f(yz, y - x) = 0$ 所确定的隐函数，其中 $f$ 具有连续的一阶偏导数，且 $f_1'(yz, y - x) \neq 0$，求 $\displaystyle \frac{\partial z}{\partial x}$ 和 $\displaystyle \frac{\partial z}{\partial y}$。

12.（6分）设函数 $u(x, y, z), \ P(x, y, z), \ Q(x, y, z), \ R(x, y, z)$ 具有一阶连续偏导数，记 $\boldsymbol{A}(x, y, z) = (P(x, y, z), Q(x, y, z), R(x, y, z))$，证明：
$$\text{div}(u\boldsymbol{A}) = u \, \text{div}\boldsymbol{A} + \text{grad} \, u \cdot \boldsymbol{A}$$

13.（6分）计算三重积分 $\displaystyle \iiint_{\Omega} \frac{x + y + z}{1 + x^2 + y^2 + z^2} \,\mathrm{d}V$ 的值，其中 $\Omega: x^2 + y^2 + z^2 \leq 1, \ z \geq 0$。

14.（6分）计算曲线积分 $\displaystyle \oint_{C} (x - y \sin x)\mathrm{d}x + (xy + \cos x)\mathrm{d}y$，其中曲线 $C$ 为有界闭区域 $D = \{(x, y) \mid 0 \leq x \leq \pi, \ 0 \leq y \leq \sin x\}$ 的边界，取逆时针方向。

15.（6分）已知函数 $f(x, y, z) = \ln(x^2 + y^2 + z^2)$。
(1)（3分）求该函数的全微分；
(2)（3分）求该函数在点 $A(1, 1, 1)$ 处沿从该点到 $B(4, 1, 5)$ 的方向的方向导数。

16.（8分）求力 $\boldsymbol{F} = (x + y^2, x - y)$ 从点 $(1, -1)$ 沿抛物线 $x = y^2$ 到原点，再沿 $y$ 轴到点 $(0, 1)$ 所做的功。

17.（8分）已知
$$[e^{-x} - f(x)] y \,\mathrm{d}x + f(x)\mathrm{d}y = 0$$
为全微分方程，其中 $f(x)$ 在 $(-\infty, +\infty)$ 内具有一阶连续导数，且 $f(0) = 1$。
(1)（4分）求函数 $f(x)$ 的表达式；
(2)（4分）求全微分方程 $[e^{-x} - f(x)] y \,\mathrm{d}x + f(x)\mathrm{d}y = 0$ 的通解。

18.（8分）
(1)（6分）求幂级数 $\displaystyle x + \frac{x^3}{3} + \cdots + \frac{x^{2n-1}}{2n - 1} + \cdots$ 的收敛域与和函数；
(2)（2分）求数值级数 $\displaystyle \frac{1}{1 \cdot 3} + \frac{1}{3 \cdot 3^3} + \cdots + \frac{1}{(2n - 1) \cdot 3^{2n-1}} + \cdots$ 的和。

19.（8分）设浴缸的排水孔是 $xOy$ 平面上半径为1cm、圆心在原点的圆孔，浴缸排水时在排水孔附近水流的流速场由下式给出：
$$\boldsymbol{v} = -\frac{y + xz}{(z^2 + 1)^2} \boldsymbol{i} - \frac{yz - x}{(z^2 + 1)^2} \boldsymbol{j} - \left(\frac{1}{z^2 + 1} + x^2\right) \boldsymbol{k} \quad (\text{cm/s})$$
(1)（3分）证明 $\text{div} \, \boldsymbol{v} = 0$；
(2)（5分）设 $S$ 为球心在原点、半径为1cm的位于 $xOy$ 平面下方的半球面，其法向指向下侧，求该流速场通过曲面 $S$ 的流量。

20.（8分）
(1)（6分）求函数 $f(x, y, z) = xyz^3$ 在条件 $x + y + z = 1 \ (x > 0, y > 0, z > 0)$ 下的极值；
(2)（2分）证明：对任意正数 $a, b, c$ 有 $\displaystyle abc^3 \leq 27\left(\frac{a + b + c}{5}\right)^5$。
