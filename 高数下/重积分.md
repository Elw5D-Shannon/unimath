# 重积分常用解法

## 一、核心思路

重积分的关键是：**选对坐标系 + 写对积分区域**。  
基本方法是将重积分化为累次积分。
**累次积分**（也称**逐次积分**）是计算多重积分的一种方法，指**按照一定顺序、逐次进行多次一元积分**的运算。
以二重积分为例：计算函数 $f(x,y)$ 在区域 $D$ 上的积分时，我们固定一个变量，先对另一个变量积分，然后再对第一个变量积分。这样就把一个二重积分转化为两个单积分。

---

## 二、常用解法速查表

| 特征                   | 方法           |
| -------------------- | ------------ |
| 区域边界清楚               | 直接化为累次积分     |
| 内层积分不好算              | 交换积分次序       |
| 区域或函数有对称性            | 对称性分析        |
| 出现 $x^2+y^2$、圆、圆环、扇形 | 极坐标          |
| 三维中圆柱、圆锥、旋转体         | 柱坐标          |
| 出现 $x^2+y^2+z^2$、球   | 球坐标          |
| 椭圆、斜线、$x+y$、$x-y$    | 一般变量代换（雅可比）  |
| 区域复杂                 | 分割区域         |
| 被积函数为1或常数            | 利用面积/体积的几何意义 |

## 三、极坐标法（二重积分）

### 变换公式
$$x = r\cos\theta,\quad y = r\sin\theta,\quad dA = r\,dr\,d\theta$$

### 何时使用
- 区域：圆、圆环、扇形、圆心不在原点的圆（如 $x^2+y^2=2ax$）
- 被积函数含 $x^2+y^2$、$\sqrt{x^2+y^2}$、$e^{-(x^2+y^2)}$ 等

### 常见区域的极坐标表示

| 区域 | 极坐标形式 |
|---|---|
| $x^2+y^2\le R^2$ | $0\le r\le R,\;0\le\theta\le2\pi$ |
| $a^2\le x^2+y^2\le b^2$ | $a\le r\le b,\;0\le\theta\le2\pi$ |
| 上半圆盘 | $0\le r\le R,\;0\le\theta\le\pi$ |
| 第一象限扇形 | $0\le r\le R,\;0\le\theta\le\frac{\pi}{2}$ |
| $x^2+y^2\le 2ax$（圆心$(a,0)$） | $0\le r\le 2a\cos\theta,\;-\frac{\pi}{2}\le\theta\le\frac{\pi}{2}$ |

### 典型例题

**例1** $\displaystyle\iint_{x^2+y^2\le1}(x^2+y^2)\,dA$
$$=\int_0^{2\pi}\int_0^1 r^2\cdot r\,dr\,d\theta
=2\pi\cdot\frac14 = \frac{\pi}{2}$$

**例2** $\displaystyle\iint_{x^2+y^2\le 2x}(x^2+y^2)\,dA$
$$=\int_{-\pi/2}^{\pi/2}\int_0^{2\cos\theta} r^3\,dr\,d\theta$$

### 常见错误
- ❌ 忘记乘 $r$（面积元）
- ❌ 角度范围错误（例如 $r=2\cos\theta$ 必须保证 $\cos\theta\ge0$）
- ❌ 混淆 $r$ 和 $\theta$ 的顺序：先定 $\theta$ 范围，再定 $r$ 范围

## 四、对称性分析

### 1. 关于坐标轴对称

| 对称轴 | 变换 | 函数满足 | 积分值 |
|---|---|---|---|
| $y$ 轴 | $(x,y)\to(-x,y)$ | $f(-x,y)=-f(x,y)$ | 0 |
| $x$ 轴 | $(x,y)\to(x,-y)$ | $f(x,-y)=-f(x,y)$ | 0 |
| 原点 | $(x,y)\to(-x,-y)$ | $f(-x,-y)=-f(x,y)$ | 0 |

若函数是偶函数（不变号），则积分等于一部分的2倍或4倍（若区域被对称轴分割）。

### 2. 关于直线 $y=x$ 对称（重点）

区域 $D$ 满足：$(x,y)\in D \Rightarrow (y,x)\in D$。

- **核心等式**：$\displaystyle\iint_D f(x,y)\,dA = \iint_D f(y,x)\,dA$
- **推论**：$\displaystyle\iint_D x^2\,dA = \iint_D y^2\,dA$，$\displaystyle\iint_D e^x\,dA = \iint_D e^y\,dA$ 等
- **反对称积分为0**：若 $f(y,x) = -f(x,y)$，则 $\displaystyle\iint_D f(x,y)\,dA = 0$  
  例如 $f(x,y)=x-y$，$f(x,y)=x^3-y^3$。

**例题**：$D$ 是单位圆盘，求 $\displaystyle\iint_D \frac{x^2}{1+x^2+y^2}\,dA$  
利用对称性：$\iint_D \frac{x^2}{1+x^2+y^2}\,dA = \iint_D \frac{y^2}{1+x^2+y^2}\,dA$  
相加得 $2I = \iint_D \frac{x^2+y^2}{1+x^2+y^2}\,dA$，再用极坐标即得。

### 3. 关于直线 $y=-x$ 对称

变换：$(x,y)\to(-y,-x)$。  
若 $f(-y,-x) = -f(x,y)$，则积分为0。  
例如 $f(x,y)=x+y$ 或 $f(x,y)=x^2-y^2$ 在 $D$ 关于 $y=-x$ 对称时积分为0。

### 4. 圆形区域的特例

圆盘 $x^2+y^2\le R^2$ 关于任意过原点的直线对称，因此：
- $\iint x\,dA = \iint y\,dA = \iint xy\,dA = \iint (x^2-y^2)\,dA = 0$
- $\iint x^2\,dA = \iint y^2\,dA = \frac12\iint (x^2+y^2)\,dA$

## 五、其他方法简述

### 1. 交换积分次序
当内层积分没有初等原函数时（如 $\int e^{y^2}dy$），通过画区域图交换 $x,y$ 的积分次序。

### 2. 柱坐标（三重积分）
$$x=r\cos\theta,\; y=r\sin\theta,\; z=z,\quad dV = r\,dr\,d\theta\,dz$$  
适用于圆柱、圆锥、旋转体及被积函数含 $x^2+y^2$ 的情形。

### 3. 球坐标（三重积分）
$$x=\rho\sin\varphi\cos\theta,\; y=\rho\sin\varphi\sin\theta,\; z=\rho\cos\varphi$$  
$$dV = \rho^2\sin\varphi\,d\rho\,d\varphi\,d\theta$$  
适用于球体、球壳、锥面与球面相交区域及含 $x^2+y^2+z^2$ 的积分。

### 4. 一般变量代换
$$x=x(u,v),\; y=y(u,v),\quad dA = \left|\frac{\partial(x,y)}{\partial(u,v)}\right| du\,dv$$  
适用于椭圆区域（如令 $x=ar\cos\theta,\;y=br\sin\theta$，则 $dA=ab\,r\,dr\,d\theta$）或含 $x+y$、$x-y$ 的组合。

### 5. 区域分割
当区域边界分段变化时，拆成若干个子区域分别积分。

### 6. 几何意义
- $\iint_D 1\,dA = \text{面积}(D)$
- $\iiint_\Omega 1\,dV = \text{体积}(\Omega)$
- 若区域对称，$\iint_D x\,dA = 0$（形心在对称轴上时）

## 六、做题流程总结

1. **看区域**：画图或想象形状，判断对称性。
2. **看函数**：是否有 $x^2+y^2$、$x^2+y^2+z^2$、奇偶性、是否可交换变量。
3. **选坐标系**：直角 / 极 / 柱 / 球 / 变量代换。
4. **定积分限**：这是最关键的一步。
5. **简化**：对称性 → 换序 → 代换。
6. **计算**：逐次积分。

> 重积分的难点不在积分本身，而在正确描述积分区域。
